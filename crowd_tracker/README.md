# 人群追踪器 (Crowd Tracker)

基于光流法的人群追踪工具，支持逐帧追踪、轨迹修正和结果导出。

## 功能特性

- 📹 **载入视频** - 支持常见视频格式 (mp4, avi, mov, mkv)，窗口最大化时画面保持比例缩放
- 🎯 **逐帧追踪** - 使用 Lucas-Kanade 光流算法追踪人头位置
- 📊 **坐标显示** - 实时显示追踪点编号和坐标
- ⌨️ **逐帧调节** - 使用左右方向键精确控制帧前进/后退
- ✏️ **轨迹修正** - 拖拽追踪点可修正位置，后续追踪基于修正后的位置计算
- ➕ **动态添加** - 在视频画面区域内点击空白处随时添加新追踪点，支持在任意帧添加
- 👁️ **显示选项** - 可独立控制追踪点、轨迹线、编号的显示/隐藏
- 💾 **结果保存** - 导出追踪轨迹为 CSV 文件

## 安装依赖


- OpenCV (opencv-python)

## 操作指南

1. **加载视频**
   - 点击"选择视频..."按钮，从文件对话框中选择视频文件
   - 视频画面会自适应窗口大小，保持原始长宽比

2. **添加追踪点**
   - 在**视频画面区域内**的空白位置用**鼠标左键**点击，即可添加新的追踪点
   - 在视频画面外点击不会产生追踪点
   - 可以在**任意帧**添加追踪点，即使视频播放到中间也可以随时添加新目标
   - 新添加的追踪点会从当前帧开始追踪

3. **播放控制**
   - 点击"▶ 播放"开始自动播放和追踪
   - 点击"⏸ 暂停"停止播放
   - 拖动进度条或输入帧号可跳转到指定位置

4. **逐帧导航**
   - 按 **←** 键：上一帧
   - 按 **→** 键：下一帧
   - 按 **空格键**：播放/暂停切换

5. **轨迹修正**
   - 按住 **Ctrl** 键，鼠标变为十字形状（或直接拖拽）
   - 按住左键拖拽追踪点到正确位置
   - 修正后的点会显示红色外圈标记
   - **重要**：修正后的位置会立即生效，下一帧的追踪将基于修正后的位置继续计算

6. **显示选项**
   - 在控制面板的"显示选项"区域，可以勾选/取消以下选项：
     - **显示追踪点**：显示/隐藏追踪点圆点
     - **显示轨迹**：显示/隐藏轨迹连接线
     - **显示编号**：显示/隐藏追踪点编号

7. **保存结果**
   - 点击"💾 保存追踪结果"按钮
   - 选择保存位置，数据将以 CSV 格式导出

## 数据格式

导出的 CSV 文件包含以下列：
- `track_id` - 追踪点编号
- `frame_idx` - 帧序号
- `x`, `y` - 坐标位置
- `is_corrected` - 是否被手动修正 (1=是, 0=否)

## 注意事项

1. 追踪精度依赖于初始点的选择和视频质量
2. 光照变化较大或遮挡严重时，追踪可能会漂移
3. 建议对重要点位进行人工修正以获得更精确的结果
